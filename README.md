# WANNADO.ROCKS — Collection Wireframe

Static wireframe prototype for the **WANNADO.ROCKS** Tattoo Art Collectibles platform.

## Pages

| File | Description |
|------|-------------|
| `index.html` | Collection Album Page with Card Detail Modal |

## Stack

- Pure HTML / CSS / Vanilla JS — zero dependencies
- Font: BDOGrotesk (with system fallback)
- Images: loaded directly from shop.wannado.rocks CDN

## Deploy on Vercel

### Option A — Vercel CLI
```bash
npm i -g vercel
vercel
```

### Option B — Vercel Dashboard (GitHub)
1. Push this repo to GitHub
2. Go to [vercel.com/new](https://vercel.com/new)
3. Import your GitHub repository
4. Framework Preset: **Other** (static)
5. Root Directory: `/` (leave as-is)
6. Click **Deploy**

No build step required. Vercel serves `index.html` automatically.

## Local Development

```bash
# Any static server works, e.g.:
npx serve .
# → open http://localhost:3000
```
